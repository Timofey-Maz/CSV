Программа считывания CSV-файлов и вывод графичекой информации на основе данных с файла.
В программе реализована функция выбора файла через диалоговое окно.

- В программе используется библиотеки numpy; matplotlib; pandas; tkinter.
- Примечание:
Программа работает только с опреленным типом CSV-файлов, с определенным названием столбцов/строк. Для тестирования программы приведен файл 3311.CSV.

Установка

- Скопируйте репозиторий
Откройте терминал и выполните:
```bash
git clone https://github.com/Timofey-Maz/CSV.git
cd CSV
```
- Установите необходимые библиотеки;
- Запустите основной скрипт:
``` bash
python UIRS.py
```
